Библейские связи — полный проект

Состав архива:
- index.html — главное веб-приложение
- bible_full_data.js — полная база книг, глав, стихов и встроенных ссылок
- bible_cross_refs_full.json — отдельный файл со всеми перекрёстными ссылками

Что внутри:
- Книг: 66
- Стихов: 31346
- Узлов исходящих ссылок: 30686
- Узлов входящих ссылок: 29933
- Всего исходящих связей: 167946

Как запустить:
1. Распакуйте архив.
2. Откройте index.html в браузере.
3. Файлы должны лежать в одной папке.

Примечание:
Файл bible_full_data.js уже содержит ссылки внутри приложения.
Файл bible_cross_refs_full.json добавлен отдельно, чтобы у вас была отдельная полная база перекрёстных ссылок.
