Проект Библейские связи — версия, переработанная по файлам:
- Основы Истины 1ч
- Основы 2

Что сделано:
- извлечены библейские ссылки из обоих документов;
- связи между стихами построены по общему контексту абзаца и по группам ссылок;
- эти связи добавлены в базу данных как документально подтвержденные;
- в интерфейсе связи с подтверждением из документов отмечаются как «док.»;
- приложение использует файл bible_full_data_doctrine.js.

Запуск:
1. Распакуйте архив.
2. Откройте файл bible_links_app_doctrine.html
